UI Navigation using React Router
================================

Navigates between a few simple pages using a sidebar and React Router.
UI made with [material-ui](https://v0.material-ui.com/#/)

Required packages:
------------------

* react
* react-dom
* react-router-dom
* mdg:geolocation
* react-responsive-carousel
* react-leaflet
* simple-flexbox
* @material-ui/core

Current pages:
--------------

* Home
* 2D Map
* Heatmap
* 3D Map
* Images
* Videos
* Settings
