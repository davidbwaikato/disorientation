UI Navigation using React Router
================================

Navigates between a few simple pages using a sidebar ([react-sidebar](https://github.com/balloob/react-sidebar)) and React Router

Required packages:
------------------

* react
* react-dom
* react-router-dom
* mdg:geolocation
* react-responsive-carousel

Current pages:
--------------

* Home
* Geolocation
* Images
* Settings
